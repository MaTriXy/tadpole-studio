# ACE Studio — Frontend

Next.js 16 + React 19 frontend for ACE Studio. See the [root README](../README.md) for full project documentation.

## Development

```bash
pnpm install
pnpm dev          # http://localhost:3000
```

## Testing

```bash
pnpm test         # Watch mode
pnpm test --run   # Single run
```

## Build

```bash
pnpm build
```
